# Assets Directory

This directory contains all image assets for the Philippine Paint Manufacturers Inc. website.

## Directory Structure

```
assets/
├── hero/
│   └── hero-background.jpg (Main hero section background image)
├── features/
│   └── (Feature icons - currently using emoji)
├── benefits/
│   ├── corrosion-protection.jpg

└── team/
    ├── ernesto-villanueva.jpg

```

## Image Guidelines

### Hero Images
- **Dimensions:** 1920x1080px minimum
- **Format:** JPG
- **Content:** Industrial/construction scenes, paint manufacturing

### Benefit Images
- **Dimensions:** 400x250px
- **Format:** JPG
- **Content:** Product showcases, application examples

### Team Photos
- **Dimensions:** 200x200px
- **Format:** JPG
- **Content:** Professional headshots

## Notes

- All images currently fall back to placeholder images if not found
- Replace placeholder images with actual company photos
- Keep file names in kebab-case format
- Optimize images for web before upload (compress to reduce file size)
